# PimpMyPi
 
You need to create a .env file in the project root with these variables:

```
# GPIO Pins
FAN_PIN=17
LED_1_PIN=23
LED_2_PIN=24
BTN_1_PIN=27
BTN_2_PIN=22

# Voltage Threshold
MIN_VOLTS=12
HIGH_VOLTS=12.2
LOW_VOLTS=10.9

# Temperature Thresholds (  C)
MIN_TEMP=35.0
MAX_TEMP=45.0

# Duty Cycle Thresholds (%)
MIN_DUTY_CYCLE=20.0
MAX_DUTY_CYCLE=100.0

# LED Settings
BLINK_INTERVAL=0.5

# Resistor Settings (Ohms)
R1_VALUE=9900.0
R2_VALUE=3333.3
```
